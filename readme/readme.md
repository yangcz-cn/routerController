## 既然要写文档，就要选取一个格式，简单又快捷
那么就是它了 md 格式 markdown
[语法说明](https://blog.csdn.net/kaitiren/article/details/38513715)

除此以外，关于标题还有等级表示法，分为六个等级，显示的文本大小依次减小。不同等级之间是以井号  #  的个数来标识的。一级标题有一个 #，二级标题有两个# ，以此类推。
